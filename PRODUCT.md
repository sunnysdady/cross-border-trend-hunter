# Product

## Register

brand

## Users
跨境电商卖家（亚马逊 / TikTok Shop / Temu / Etsy 等），多为中文母语的运营、选品、老板。使用场景：每天花几分钟扫一眼「今天有什么不该火却火了的爆款」，作为选品和跨境套利的决策线索。他们要在信息里快速判断「这个值不值得跟」，所以可信度 > 炫技。

## Product Purpose
每日自动扫描全球电商平台，捕捉异常爆款信号，并给出 VOC（消费者声音）与商业可行性分析。本质是一份「每日趋势情报简报」，由 GitHub Actions + Claude 全自动生成、Vercel 部署。成功标准：卖家信任这里的判断，愿意每天回来看，并据此做选品动作。

## Brand Personality
敏锐、可信、有态度的趋势分析师。像一个懂行的买手朋友每天给你递情报：专业但不端着，偶尔毒舌。三词：敏锐、可信、有记忆点。情绪目标：让人觉得「这家是认真在分析，不是在堆词」。

## Anti-references
- 霓虹赛博朋克 / 渐变发光科技风——选品数据品类的 AI-slop 第一反射，炫光削弱数据可信度。
- 渐变文字标题、满屏粒子背景、毛玻璃卡片——装饰性无意义特效。
- 千篇一律的等宽卡片网格 + 图标-标题-正文模板。
- 把「严肃数据」做成廉价营销落地页的浮夸感。

## Design Principles
1. 数据即权威——让数字、来源、情绪占比成为视觉主角，可信度第一。
2. 克制的惊喜——「盲盒」是品牌记忆点，保留为锦上添花，但绝不拿它当内容可见性的开关。
3. 层次靠排版，不靠特效——用字号/字重/留白建立秩序，而非边框和发光。
4. 每日一色——每天一个克制的主色作为唯一强调色，承载当天身份，而非霓虹渐变汤。
5. 默认可读——内容默认完整呈现，动效只增强已可见的内容。

## Accessibility & Inclusion
WCAG 2.1 AA：正文对比度 ≥4.5:1，大字 ≥3:1。所有交互元素为语义化 button/可键盘操作，带 aria 状态。尊重 prefers-reduced-motion（提供淡入或瞬时替代）。中文为主，字体栈需覆盖 CJK。
